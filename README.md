This repository contains my solutions to the problem sets and the final project from the **CS50's Introduction to Programming with Python** course offered by Harvard University.

##  About the Course
CS50P is a comprehensive introduction to programming using Python. It covers topics from basic variables and functions to more advanced concepts like regular expressions, object-oriented programming (OOP), and file I/O.
